
This dataset contains information about movies, including their titles, genres, release details & audience ratings.

- Number of rows: 1000
- Number of columns: 9

Columns:
1- MovieID (int64): Unique identifier for each movie
2- Title (object): Name of the movie
3- Genre (object): Movie genre
4- ReleaseYear (int64): Year of release
5- ReleaseDate (datetime64[ns]): Full release date (format: DD-MM-YYYY)
6- Country (object): Country of production
7- RottenTomatoesScore (int64): Rotten Tomatoes rating (%)
8- NumVotesIMDb (int64) : Number of IMDb votes
9- NumVotesRT (int64): Number of Rotten Tomatoes votes


The dataset contains no missing values and no duplicate records.

Source: Kaggle


This dataset is used for exploratory data analysis (EDA), including examining distributions, comparing ratings across categories & identifying relationships between variables such as ratings, votes, and release dates.
